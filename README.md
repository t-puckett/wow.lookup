# wow.lookup
1. CD wow.lookup/web
2. npm install
3. npm run build
4. npm run dev


# backend
when you are in backend vial terminal rin the following to start the virtual environment
1. Install python 
2. Update Pip to latest
3. Change content policy if applicable
4. cd in top wow.lookup/backend
5. Run 'python -m venv venv'
6. cd into backend/venv/Scripts
7. Run Powershell command ./Activate.ps1
8. Run pip install -r requirements.txt
